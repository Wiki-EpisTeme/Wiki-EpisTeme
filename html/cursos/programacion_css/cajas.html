Modelo de caja.

modelo de caja: es la forma en que CSS ve los elementos HTML y ¬øcomo los ve? como si fueran cajas con las siguientes propiedades:

  1. contenido (content): el contenido (texto) del elemento HTML.tomando encuento sus dimencionnes(alto y ancho)
  2. borde (border): lo que delimita cada uno de los elementos
  3. relleno (padding):son las distancas internas. es decir la distancas ente el borde y el contenido
  4. margen (margin): son las distancias esternas (la distancia entre el elemento HTML y sus elementos hermanos o padres)
Dentro del modelo de caja hay que considerar que una caja tendra 4 lados
  1. arriva(top)
  2. derecha(right)
  3. abajo(bottom)
  4. izquierda(left)

Cajas de Bloque o de Linea
                    
  • Cajas de linea
    • 
                    -ocupan el espacio necesario para mostrar su contenido.
                    -no tinen dimenciones modificables(alto y ancho).
                    -permitenotros elementos a su lados.
                    -padding y margin solo empujan a elementos a elementos adyacentes en horizontal,nunca en vertical.
                    
  • Caja de bloque
    • 
                    -ocupan todo el ancho disponible.
                    -tienen todo el ancho disponible lo que genera saltos de lineas.
                    -no permiten otros elementos a su lado (aunque especifique un ancho,sigue ocupando todo el espacio)
