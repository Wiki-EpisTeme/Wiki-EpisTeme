etiquetas semanticas

"div" no es una etiqueta semantica

"header"es cabecera de un sitio web o se una seccion

"footer"pie de pagina de un sitio web o de una seccion

"main" define la seccion principal, solo puede existir un "main"

"nav"representa una navegacion

"article"representa una seccion autocontenido(que por si sola se esplica)

"aside" representa contenido complementario o secuencia(o publicidad)

"section" representa un bloque de contenido

"address" representa una direccion o direccion de contacto

etiquetas de bloque vs etiquetas de linea

"div" es la etiqueta de bloque por excelencia, es aquella que genera santos de linea con sus etiquetas hermanas
"span"es la etiqueta de linea por excelencia, solo ocupa el estacio necesario que tiene su contenido

formas de estilo de cascada

"style" se utiliza para agrega color o tipografia a todo la pagina, y se puede o tiene que utilizar en la cabeza del documento
